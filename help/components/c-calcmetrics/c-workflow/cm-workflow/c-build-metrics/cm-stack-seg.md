# 堆叠和替换区段

您可以在计算量度生成器中栈叠和替换区段。

## 堆叠区段 {#section_3FC1118FEE2B40B7BB90F29C8521AF53}

在“定义”画布中，只需将新区段拖放到现有区段旁边：

![](assets/cm_stack_seg.png)

## 使用一个区段替换另一个 {#section_3DA899DF522A44D7A583C5DC8C0A033E}

在“定义”画布中，只需将新区段放在现有区段的上方即可：

![](assets/cm_replace_seg.png)
