# 上次访问间隔天数

“上次访问后的天数”维度测量访客的当前点击与其上次访问（如果有）之间所经过的时间。 此维度有助于您了解访客在访问您的网站后所做的行为。 示例包括：

* 用户重新访问网站的频率是多少？
* 回访频度与转化有何关联？重复购买者是否经常访问？
* 点击活动的用户是否经常返回？

首次访客未包含在此维中。

## 用数据填充此维

此维度适用于所有实施。 如果报表包包含数据，则此维有效。

## 维项

维项目包括访客上次访问与当前点击之间的天数。 每个天数是单独的维度项目， `"Same day"` 访客上次访问和当前点击发生在同一天。
