# Analytics 维度兼容性

本页列出了各自 Analytics 功能中支持的维度。

>[!NOTE]
>
>此列表中删除了自定义变量名称、分类和访客属性。这些维度项目特定于单个报表包。

>[!NOTE]
>
>在 Analytics 工具对类似维度使用不同术语时存在一些重叠。例如，Reports &amp; Analytics 使用 `browserwidth`，而 Analysis Workspace 使用 `browserwidthbucketed`。
