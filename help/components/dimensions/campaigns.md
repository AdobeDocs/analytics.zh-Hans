# 营销活动

* 在某些报表包中自动创建的[分类](../classifications/classifications-overview.md)。 它基于[跟踪代码](tracking-code.md)维度。
* [跟踪代码](tracking-code.md)维度本身，它会使用[促销活动](/help/implement/vars/page-vars/campaign.md)实施变量。
