# 周

“周”维度报告给定度量出现的周。 第一个维度项目是日期范围内的第一周，最后一个维度项目是日期范围内的最后一周。 此维度对于趋势报表至关重要，因为它允许您查看一段时间内的指标。

## 用数据填充此维

此维度适用于所有实施。 如果报表包包含数据，则此维有效。

## 维项

维项目包括一周中第一天的日期（月、日和年）。
