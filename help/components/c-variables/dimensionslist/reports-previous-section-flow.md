# 上一区域流量

“上一区域流量”报表与“上一页面流量”报表类似。它显示“网站区域”（相关网页组）的数据。如果某个页面包含在多个网站区域中，此报表将显示所有网站区域的数据。

例如，一家在线零售商可能会将一些网站区域用于展示产品，另外一些网站区域用于展示产品品牌。在这种情况下，产品网页可能包含在多个网站区域中。虽然某个产品页面只查看过一次，但上一区域流量报表会显示与该页面关联的每个网站区域的页面查看情况。
