# 上次访问间隔天数

“上次访问间隔天数”是一个随 Analysis Workspace 一起提供的现成 Analytics 维度。

它可以确定自用户上次访问您网站后过去的天数。如果将点击计为新访问，它会查看上次点击的时间戳（如果有的话），并根据上次点击发生后过去的天数来对访问进行分段统计。

此回访频度数据可用于解答以下问题：

* 用户再次访问我的网站的频率是多久？
* 回访频度与转化有何关联？重复购买者是否经常访问？
* 点击我的促销活动的用户是否经常回访？
