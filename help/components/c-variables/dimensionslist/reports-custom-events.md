# 自定义事件

您希望访客在您的网站上完成的转化操作。这些操作可能是注册、订阅和填写商机表单、开始聊天、购买、预订或完成调查。

由于每个报表包都略有不同，因此每个客户对此报表包的使用方式也不尽相同。自定义事件报表可用作显示事件发生次数的计数器。例如，如果将 **[!UICONTROL event1]** 设置为计算文档的下载次数，则事件 1 的自定义事件报表会显示该事件（或下载）发生的总次数。你可使用多个自定义事件报表。
