# 购物车

显示指定时间段内打开的购物车数目。您可以运行报表以分析购物车查看、加货、减货和结账。购物车通常在客户选择购买项目时打开，但在没有项目时也能打开它。

“[!UICONTROL 购物车]”可用于：

* 确定网站上已打开购物车的模式或数量的高值或低值。
* 详细检查特定时间段，以了解对打开购物车有特别贡献的量度的详细信息。
