# JavaScript 版本

按移动设备支持的 Java 版本对移动设备进行分组。JavaScript 版本报表通过设备支持的 Java Specification Request (JSR) 来识别 Java 版本。

有关 JSR 的详细信息，请访问 [Java Community Process](https://jcp.org/en/jsr/overview) 网站。
