# 浏览器高度

专指浏览器窗口中数据的垂直距离。浏览器高度维度不包括工具栏、菜单、按钮等项目。

Adobe Analytics 仅使用来自某访问第一次点击的浏览器高度和宽度。其余的点击不会得到同一访问的属性。

有关更多信息，请参阅 [浏览器宽度/高度](/help/components/c-variables/dimensionslist/browser-width.md).
