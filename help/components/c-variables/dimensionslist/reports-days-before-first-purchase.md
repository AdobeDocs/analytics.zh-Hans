# 首次购买间隔天数

显示从客户首次访问网站到最终决定购买之间的间隔天数。

例如，访客在首次访问网站的一天后购买了商品，那么在该访客 Cookie 有效期的剩余时间内，报表的“1 天”**&#x200B;行项目中将显示相关的任何访问或事件。
