# 查找方法

显示由哪方接收转化成功事件的信用。例如，如果某个搜索引擎引荐一个访客到您的网站并购物，“查找方法”可指定该搜索引擎如何接收引荐的信用。

See [Finding Methods](/help/admin/admin/finding-methods.md) for configuration information.
