# 点击类型

“点击类型”维度表示在收集点击时，应用程序是处于前台还是后台。

此维度由 SDK 自动收集。SDK 4.13.6 及更高版本支持此维度。

如果选中“禁用计算后台点击量的旧版报告和属性”，则后台点击量将仅显示在[虚拟报告套件](../../../components/vrs/vrs-mobile-visit-processing.md#concept_EC51308E4FD14E149F1B5D63C0AB34BD)。
