# PathFinder

PathFinder 报表可让您进一步将整个路径剖析为片段，从而生成有助于优化网站的精确模式。

此报表的强大功能在于过滤器向导，通过该向导，您可以指定用于生成报表的选择标准。您可以使用过滤功能分析路径片段，查询以下几种路径片段：以特定页面开头，以特定页面结尾，或以一个页面开头并以另一个页面结尾。
