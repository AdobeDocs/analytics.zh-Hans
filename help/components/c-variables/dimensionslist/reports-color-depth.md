# 颜色深度

按支持的颜色数目对移动设备点击量进行分组。此报表/维度显示经由移动设备访问网站的访客总人数，并按移动设备中配置的颜色数目对他们进行分组。例如，若访客的移动电话支持 24 色，则报表将增加与 24 色对应的行项目。
