# 颜色深度

按支持的颜色数目对移动设备点击量进行分组。此报表/维度显示经由移动设备访问网站的访客总人数，并按移动设备中配置的颜色数目对他们进行分组。例如，如果访客的手机支持24种颜色，则报表会增加与24种颜色对应的行项目。
