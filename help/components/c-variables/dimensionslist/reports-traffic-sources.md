# 流量源

显示有关向您的网站导引流量的网络位置信息。您可以看到哪些搜索引擎和域外网站向您的网站导引访客。

## 流量源{#topic_6718F8EFD5984DC5839B9E8F6CC45EDA}

显示有关向您的网站导引流量的网络位置信息。您可以看到哪些搜索引擎和域外网站向您的网站导引访客。

此菜单中的报表分为以下三个基本类别：

* 搜索引擎
* 搜索关键词
* 反向链接和反向链接域名
