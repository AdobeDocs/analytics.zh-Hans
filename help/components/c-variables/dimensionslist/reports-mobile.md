# 移动设备

显示有关从移动设备访问网站的信息。 移动报告通过收集用于访客网站的移动设备的相关数据来改进视图识别。 移动报告可以识别设备的制造商和功能，如屏幕大小、视频、音频和Cookie支持等指标。 移动报告可用于所有报告套件。

## 移动设备 {#topic_D306EA4558194488AC47A45B9C570150}

显示有关从移动设备访问网站的信息。 移动报告通过收集用于访客网站的移动设备的相关数据来改进视图识别。 移动报告可以识别设备的制造商和功能，如屏幕大小、视频、音频和Cookie支持等指标。 移动报告可用于所有报告套件。

>[!NOTE] 移动报表仅显示移动 Web 流量。使用本机iOS、Android和其他SDK收集的移动应用程序数据显示在移动应用程序报告中，并且仅在“管理工具”中启用时才显示。
