# 维度描述

Reports &amp; Analytics 中可用的预定义报表的定义，以及其他 Analytics 工具（如 Analysis Workspace）中可用维度的定义。

维度是可以在报表中查看、划分和比较的量度数据的说明或特性。它们是非数字值和日期，可细分或是原始报告类型度量的分类。
