# 信息服务

按支持的新闻服务（渠道 I/B、EZ 新闻 Flash、W+INFO 等）对移动设备进行分组。当访客使用支持新闻服务的移动设备访问网站时，信息服务报表至少递增 1。如果移动电话支持多种新闻服务，报表中的这些组将递增。这样，群组总数可能大于报表底部显示的总数。
