# 购物车减货

| 使用 | 描述 |
|---|---|
| 流量 | 不适用 |
| 转化 | 设置[!UICONTROL 购物车减货]事件的实例。发生在从购物车移除商品时。 |
| Ad Hoc Analysis | 设置[!UICONTROL 购物车减货]事件的实例。发生在从购物车移除商品时。 |
