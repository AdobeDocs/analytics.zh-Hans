# 登录

登录次数表示作为访问中第一个值所捕获到的给定值的次数。登录在每次访问中只能发生一次。但是，如果未定义变量，则它不一定是首次点击。

登录页面包含一个访问划分范围，表示这些页面在一次访问的所有点击中一直存留。请参阅[划分和分段容器](https://marketing.adobe.com/resources/help/en_US/sc/user/?f=c_Breakdown_and_segmentation_containers)以了解详细信息。
