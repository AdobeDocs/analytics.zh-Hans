# 跳出次数

由单个服务器调用组成的访问。 例如，如果访客未以向Adobe发送数据(如单击链接或视频开始)的方式与页面交互，则单页访问是一次跳出。 如果在一次访问中收到的点击次数超过一次，则不会计算跳出次数。
