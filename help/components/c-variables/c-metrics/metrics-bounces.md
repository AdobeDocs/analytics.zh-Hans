# 跳出次数

只包含一次服务器调用的访问。例如，如果访客没有与页面进行需要向 Adobe 发送数据的交互操作（如单击链接或启动视频），则一次页面访问就是一次跳出。如果访问中收到的点击次数超过一次，那么将不会计为一次跳出。
