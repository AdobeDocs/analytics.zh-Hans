# 跳出次数

只包含一次服务器调用的访问。例如，如果访客未以向Adobe发送数据(如单击链接或视频开始)的方式与页面交互，则单页访问是一次跳出。 如果访问中收到的点击次数超过一次，那么将不会计为一次跳出。
