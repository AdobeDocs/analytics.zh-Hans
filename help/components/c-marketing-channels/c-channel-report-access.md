# 报表访问

介绍如何运行营销渠道报表、授予有限管理员用户权限，以及授予用户群组对报表的访问权限。

为了查看营销渠道报表，需要授予用户群组适当的权限。Adobe Experience Cloud 的权限系统以群组为基础，这表示用户所在群组决定了对报表的访问权限。
