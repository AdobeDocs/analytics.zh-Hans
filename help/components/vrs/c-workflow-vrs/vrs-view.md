# 查看虚拟报告包信息

单击报表包名称旁的 i（信息）图标即可获得关于报表包的信息。

## 在报表包选择器中 {#section_74E43B60C1CA4180B5ACA57574C1FA0F}

单击报表包选择器中虚拟报表包旁边的信息图标可提供以下信息：

* 父报表包的名称。
* 应用于它的任何区段的名称。
* 已应用区段的报表包的简单预览。
* 独特访客总数。
