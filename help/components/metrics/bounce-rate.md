# 跳出率

“跳出率”量度会显示包含一次点击的访问量与该页面上的访问量之间的比率。利用此量度，您可以了解哪个维度项目的跳出率最高，或查看一段时间内网站的累计总跳出率。高跳出率可以揭示网站或应用程序的问题，例如：设计不良、加载时间缓慢、内容不相关或访客意图与页面内容不匹配。

## 如何计算此指标

此量度使用公式计算 [`Bounces`](bounces.md) `divided by` [`Entries`](entries.md).
