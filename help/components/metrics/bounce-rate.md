# 跳出率

“跳出率”量度会显示包含一次点击的访问量与该页面上的访问量之间的比率。利用此量度，您可以了解哪个维度项目的跳出率最高，或查看一段时间内网站的累计总跳出率。

## 如何计算此量度

使用公式 [`Bounces`](bounces.md)` divided by `[`Entries`](entries.md) 计算此量度。
