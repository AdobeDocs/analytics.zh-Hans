# 跳出率

“弹出率”指标显示包含一次点击的访问量与该页面上的访问量之比。 您可以使用此指标了解哪些维度项目的弹出率最高，或查看一段时间内您网站的累计总弹出率。

## 如何计算此度量

此度量使用公式计算 [`Bounces`](bounces.md)` divided by `[`Entries`](entries.md)。
