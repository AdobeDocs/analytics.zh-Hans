# 人员

“人员”量度根据Adobe的设备图表计算人员或设备组的数量。 它更准确地表示唯一访客 [的数量](unique-visitors.md)。

See the [People metric](https://docs.adobe.com/content/help/zh-Hans/device-co-op/using/data/people.html) in the Device Co-op user guide.
