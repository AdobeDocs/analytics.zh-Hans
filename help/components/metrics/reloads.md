# 重新载入

“重新载入”量度显示重新载入过程中维度项目的出现次数。访客刷新其浏览器是触发重新载入的最常见方式。

## 如何计算此量度

此量度计数 [`Page`](../dimensions/page.md) 维度包含与上一次点击相同的值的点击次数。
