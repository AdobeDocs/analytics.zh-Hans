# 重新载入

“重新加载”度量显示重新加载过程中维度项的出现次数。 刷新浏览器的访客是触发重新加载的最常见方式。

## 如何计算此量度

此度量计算维度包含与上次 [`Page`](../dimensions/page.md) 点击相同值的点击数。
