# 退回

“弹回次数”指标显示仅包含一次点击的访问次数。 您可以使用此指标了解您获得的弹回与非弹回流量的比率，查看导致访客弹回的顶部进入页面，或查看弹回总数随时间的趋势。

## 如何计算此度量

如果访问只包含一次点击，则点击被视为退回。

>[!NOTE] 如果访客同时触发页面视图和链接跟踪调用，则该访问不被视为退回。 请考虑改 [用单页访问](single-page-visits.md) 。
