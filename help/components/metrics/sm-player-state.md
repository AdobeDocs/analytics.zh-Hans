# 流媒体服务播放器状态跟踪量度

流媒体服务播放器状态跟踪量度为通过流媒体收集库收集的数据提供补充报表功能。 这些量度需要&#x200B;**[!UICONTROL 适用于流媒体的Adobe Analytics加载项]**。 有关详细信息，请联系您的Adobe客户团队。

若要使用这些量度，请在[[!UICONTROL 媒体报告]](/help/admin/tools/manage-rs/edit-settings/media-management.md)下为报表包启用&#x200B;**[!UICONTROL 播放器状态跟踪]**。

可以使用以下量度：

* [[!UICONTROL 受隐藏式字幕影响的流数量]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-streams-impacted)
* [[!UICONTROL 隐藏式字幕计数]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-count)
* [[!UICONTROL 隐藏式字幕总时长]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-total-duration)
* [[!UICONTROL 受全屏影响的流数量]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-streams-impacted)
* [[!UICONTROL 全屏计数]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-count)
* [[!UICONTROL 全屏总时长]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-total-duration)
* [[!UICONTROL 受聚焦影响的流数量]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-streams-impacted)
* [[!UICONTROL 聚焦计数]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-count)
* [[!UICONTROL 观看中总持续时间]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-total-duration)
* [[!UICONTROL 受静音影响的流数量]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-streams-impacted)
* [[!UICONTROL 静音计数]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-count)
* [[!UICONTROL 静音总持续时间]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-total-duration)
* [[!UICONTROL 受画中画影响的流数量]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-streams-impacted)
* [[!UICONTROL 画中画计数]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-count)
* [[!UICONTROL 画中画总持续时间]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-total-duration)
