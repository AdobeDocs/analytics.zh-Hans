# 购物车加货

“购物车加货” [量度](overview.md) 显示访客将商品添加到购物车的次数。 如果您想要了解转化漏斗中客户对某个产品非常感兴趣并将其添加到购物车，此量度非常有用。

## 如何计算此量度

此指标计算 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 变量中存在 `scAdd` 的点击数。

请参阅 [购物车](carts.md) 用于量度比较。
