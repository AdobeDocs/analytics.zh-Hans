# 购物车视图

“购物车视图”量度显示访客查看其购物车的次数。 当您想要了解客户视图购物车中内容时，此指标非常有用。

## 如何计算此度量

此度量计算变量中存 `scView` 在的点击 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 数。
