# 购物车查看

“购物车查看次数” [量度](overview.md) 显示访客查看其购物车的次数。 当您想要了解转化漏斗中客户查看其购物车中的内容时，此量度非常有用。

## 如何计算此量度

此指标计算 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 变量中存在 `scView` 的点击数。

请参阅 [购物车](carts.md) 用于量度比较。
