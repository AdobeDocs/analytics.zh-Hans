# 结账

“结帐”[量度](overview.md)显示访客启动结帐流程的次数。 当您想要了解转化漏斗中客户将产品添加到购物车并准备购买时，此量度非常有用。

## 如何计算此指标

此指标计算 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 变量中存在 `scCheckout` 的点击数。
