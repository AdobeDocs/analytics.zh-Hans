# 结账

“结帐”度量显示访客启动结帐过程的次数。 当您想要了解客户拥有购物车产品并准备购买的转化漏斗的一部分时，此指标会很有帮助。

## 如何计算此度量

此度量计算变量中存 `scCheckout` 在的点击 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 数。
