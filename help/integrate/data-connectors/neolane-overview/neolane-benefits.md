# Key Benefits and Features{#key-benefits-and-features}

此集成包括以下主要优点：

* 将电子邮件营销和分析数据整合到一个报告界面中。
* 通过转化率和对收入和网站成功的贡献优化电子邮件营销活动。
* 根据动态营销细分，对关键访客和市场细分进行营销。
