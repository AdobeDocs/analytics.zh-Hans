# 主要优点{#key-benefits}

列出Adobe Analytics与Responsys交互集成的主要益处。

* 将电子邮件营销和分析数据整合到一个报告界面中。
* 通过转化率和对收入和网站成功的贡献优化电子邮件营销活动。
* 根据购物车废弃、产品购买、产品视图、跨越当前和历史数据(激活前90天)对关键访客和市场细分进行营销。
