# Javascript 库文件

JavaScript 库文件将在首次加载之后，被放入用户浏览器的缓存中，这可以有效限制需要下载的数据量。

针对页面的变量应置于 HTML 片段中。其他所有变量都应置于 JavaScript 库文件中。
