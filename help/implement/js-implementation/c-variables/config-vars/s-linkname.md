# s.linkName

自定义、下载或退出链接报告中将显示的名称。 从 100 个字符处截断。可设置为任何字符串。此为图像请求中的 `pev2` 参数。

如果通过 `linkType` 设置，则图像请求将作为下载、自定义或退出链接发送.
