# s.linkName

该名称将在自定义、下载或退出链接报表中显示。从 100 个字符处截断。可设置为任何字符串。此为图像请求中的 `pev2` 参数。

如果通过 [`linkType`](https://docs.adobe.com/content/help/en/analytics/implementation/javascript-implementation/variables-analytics-reporting/config-var/linktype.html) 设置，则图像请求将作为下载、自定义或退出链接发送。
