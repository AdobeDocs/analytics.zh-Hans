# trackingServerSecure

 变量用于第一方 Cookie 实施，以指定写入图像请求和 Cookie 的域名。


<!-- 

trackingServerSecure.xml

 -->

用于安全页面。如果 *`trackingServerSecure`* 未定义，SSL 数据将转至 *`trackingServer`*.

| 最大大小 | 调试程序参数 | 填充报表 | 默认值 |
|---|---|---|---|
| 不适用 | 不适用 | 不适用 | "" |
