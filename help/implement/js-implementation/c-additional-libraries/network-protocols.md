# 移动网络协议

WAP 和 I-Mode 是当今采用的两种主要协议或标准。WAP 主要用于美国，而 I-Mode 常用于日本和欧洲。

必须分别针对不同的协议单独设计网站。不需要针对每种协议自定义 Adobe 图像标记。
