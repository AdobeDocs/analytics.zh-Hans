# 从页面模板移动到页面模板

如果您的网站有多种类型的页面或页面模板，则可使用路径功能来了解访客如何在各种类型之间移动。

您首先需要在专为此目的而设置的 prop 中捕获页面类型或模板。然后，让 Adobe [!DNL Customer Care] 来启用该 prop 上的路径功能。现在您便可以查看页面模板路径报表，以了解用户如何在网站上的模板之间移动。
