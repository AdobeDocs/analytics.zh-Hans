# Util.cookieRead

获取 Cookie 的值。

**语法：**

```
s.Util.cookieRead(key)
```

**参数：**

| 参数 | 描述 |
|---|---|
| key | （必需）写入 Cookie 值的键。 |

**返回结果：**

Cookie 值，或者如果找不到 Cookie，则返回空字符串。

**示例：**

```js
var myCookie = s.Util.cookieRead("my_cookie");
```
