# 量度

量度是有关访客活动的量化信息，包括查看、点进次数、重新载入、平均逗留时间、件数、订购和收入等。

量度是报表的基础，可帮助您查看并理解数据关系。您可以利用它们，对网站中的不同数据集执行对照比较。
