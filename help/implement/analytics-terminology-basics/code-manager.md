# 代码管理器

代码管理器可让您下载适用于 Web 和移动平台的数据收集代码。

**[!UICONTROL Analytics &gt; 管理员 &gt; 代码管理器]**。

下载库后，您必须配置代码，以便将数据发送到正确的跟踪服务器和报表包。您可以在[开发人员和实施](https://marketing.adobe.com/resources/help/zh_CN/reference/developer.html)中找到其他实施资源。
