# 区段

区段是数据的自定义子集，或者是由您创建的规则所过滤的数据。

区段基于以下项：

* 点击量
* 访客
* 访问

请参 [阅组件用户指南](/help/components/c-segmentation/seg-home.md) 中的分析分段。
