# dc

> [!IMPORTANT] 此变量已停用。 请改 `trackingServer` 用。

在Adobe Analytics的先前版本中，Adobe要求您指定要将数据发送到哪个数据中心。 将点击发送到错误的数据中心会导致数据丢失。

Adobe通过允许任何实施将点击发送到来改进此体验 `sc.omtrdc.net`。 不再需要指定数据中心。
