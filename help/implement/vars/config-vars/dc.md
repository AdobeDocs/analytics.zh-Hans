# dc

>[!IMPORTANT] 此变量已停用。请改用 [`trackingServer`](trackingserver.md)。

在早期版本的 Adobe Analytics 中，Adobe 要求您指定要将数据发送到哪个数据中心。将点击发送到错误的数据中心会导致数据丢失。

通过允许所有实施将点击发送到 `sc.omtrdc.net`，Adobe 改进了此体验。不再需要指定数据中心。
