# 日志文件

日志文件是一个文本文件，服务器使用该文件存储有关何时从网站检索文件的数据。收集的部分信息包括日期和时间、提供的 URL、IP 地址、请求的状态等。

保存和查看日志文件的快捷方法是：

1. 单击“帮助”****>“关于 Ad Hoc Analysis”****

1. 单击“Ad Hoc Analysis”****&#x200B;徽标。
1. 单击“保存日志文件”****。

>[!NOTE]
>
>使用此过程保存的日志适用于当前会话。
