# 布局 - 请求向导第 2 步

通过“请求向导: 第 2 步”窗口，您可以配置输出的报表布局、标题格式和维度过滤。您可以使用引导布局配置请求，这与创建 Excel 数据透视表类似。

或者，您可以手动选择要在其中显示报表数据的单元格，只需使用[自定义布局](/help/analyze/report-builder/layout/configure-the-custom-layout.md)即可。
