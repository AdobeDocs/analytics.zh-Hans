# 在同一请求中组合使用流量和商务量度

您可以在一个请求中组合使用流量和商务量度类型，而不是创建单独的请求。

流量和商务量度在“请求向导：第2步”的同一量度类别中显示。 已分组类别带有标签 **[!UICONTROL 标准]**.
