# 在同一请求中组合使用流量和商务量度

现在您可以在同一请求中组合使用这两种量度类型，而不必分别创建不同的请求。

现在，在“请求向导”的第 2 步中，流量和商务量度显示在同一个量度类别中。以前，这两个量度分别显示在“商务”类别和“流量”类别中。

新的编组类别标记为&#x200B;**[!UICONTROL 标准]**。
