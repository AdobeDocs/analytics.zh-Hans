# 受保护的工作表元素

如果您通过锁定单元格保护工作表，则无法创建、编辑或刷新请求。在使用请求管理器之前，请取消工作表中的所有保护。

如果您正在刷新请求，并且尝试在刷新请求过程中编辑单元格，Excel 将显示一条错误消息，告诉您电子表格当前处于受保护状态。刷新请求过程完成后，您可以访问电子表格，并且系统会取消临时电子表格保护。
