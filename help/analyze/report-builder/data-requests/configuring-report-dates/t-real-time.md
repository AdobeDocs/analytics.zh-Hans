# 配置实时请求

用于描述如何配置实时请求日期的步骤。

1. 确保在市场营销 Reports &amp; Analytics 的[管理工具](https://docs.adobe.com/content/help/en/analytics/admin/admin-tools/real-time-reports/t-realtime-admin.html)中已启用实时报告。
1. 在上 [!UICONTROL Request Wizard: Step 1]，单击 **[!UICONTROL Real-Time Report]** > **[!UICONTROL `<report type>`]**

   例如，选择“流量报表”。When you select a real-time report type, the [!UICONTROL Select Time Range] options display.

1. 以分钟或小时为单位选择一个时间范围。

   ![步骤结果](assets/real_time_select_date.png)

   实时报告仅适用于最近 20 个小时。对于粒度，您的选择范围介于 1 分钟到 30 分钟之间。
1. Click **[!UICONTROL Next]** and continue [configuring the request layout](/help/analyze/report-builder/layout/layout.md).
