# 单个开始日期或结束日期

开始日期和结束日期的缩写。

* cd-1d = 昨天
* cm-1d = 上个月的最后一天
* cw-1w = 上周的第一天
* cw+1w-1d = 当前周的最后一天
* cd-2d = 两天前
* cw-2w = 两周前的那一周的第一天

>[!NOTE]
>
>如果将日期设置为将来的日期，则不会返回值。
