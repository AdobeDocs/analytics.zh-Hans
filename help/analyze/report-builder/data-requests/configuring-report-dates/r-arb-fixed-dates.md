# 请求向导定义 - 固定日期

Report Builder 中固定日期的定义。

| 字段 | 定义 |
|--- |--- |
| 此报表包 | 使保存的日期范围仅可用于当前报表包。 |
| 所有报表包 | 使保存的日期范围可用于登录用户可访问的所有报表包。 |
| 输入参数名称 | 为保存的日期键入名称。您可以将此日期用作Report Builder中的模板。 在可应用已保存日期的所有位置，都会显示此名称。更改模板的名称并不影响先前创建的数据请求中使用的日期配置。 |
