# 滚动日期

说明如何为请求自定义滚动日期的步骤。

1. 在“[!UICONTROL 请求向导: 第 1 步]”中，选择&#x200B;**[!UICONTROL 滚动日期]**。
1. 单击&#x200B;**[!UICONTROL 滚动]**&#x200B;链接以选择滚动类型，然后使用日期链接配置滚动日期。

   “[!UICONTROL 滚动日期]”的设置根据您的选择而发生变化。例如，选择“[!UICONTROL 每周滚动]”时，Report Builder 会提供可选作滚动模式的特定于周的设置。

1. 根据需要配置高级选项。
