# 选择报表类型

为数据请求选择基本报表类型的步骤。

1. 导航到“[!UICONTROL 请求向导: 第 1 步]”窗口。
1. 从&#x200B;**[!UICONTROL 报表包]**&#x200B;菜单中选择一个报表包。
1. 为数据请求选择基本报表类型。
