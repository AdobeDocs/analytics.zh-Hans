# 选择一个报表包

帮助您在 Excel 中选择报表包的步骤。

1. In Excel, click **[!UICONTROL Add-Ins]**.
1. In the toolbar, click **[!UICONTROL Create]**.
1. 在窗 [!UICONTROL Request Wizard: Step 1] 口中，选择报表包下 [!UICONTROL Report Suite]。
