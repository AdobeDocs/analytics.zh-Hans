# 在 Excel 中通过现有请求选择报表包

帮助您从现有请求选择报表包的步骤。

1. 在 Excel 中，单击&#x200B;**[!UICONTROL 外接程序]**。
1. 在工具栏中，单击&#x200B;**[!UICONTROL 创建]**。
1. 在“[!UICONTROL 请求向导: 第 1 步]”窗口中，单击&#x200B;**[!UICONTROL 范围选择器]** ( ![](assets/select_cell_icon.png) ）。
1. 使用“[!UICONTROL 范围选择]”窗体定位单元格。
1. 继续创建请求。
