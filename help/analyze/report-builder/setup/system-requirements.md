# 系统要求

系统建议和先决条件。

* Windows XP 或更高版本
* 32 位或 64 位 Microsoft Excel 2007 或更高版本
* 不支持 OS X

**先决条件**

* 您组织的合同包含适用于 Excel 的 Report Builder。
* 您的用户名包含在预定义的 Report Builder 用户组中。This requirement includes [!DNL Analytics] administrators.
* 用户运行的必须是 Windows，并以管理员的身份登录。
