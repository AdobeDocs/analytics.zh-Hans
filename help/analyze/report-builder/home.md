# Report Builder 是什么？

Adobe Report Builder 是 Microsoft Excel 的一个加载项。它允许您从Adobe Analytics数据构建自定义请求，您可以将其插入到Excel工作表中。 请求可动态引用工作表中的单元格，并且您可更新和自定义 Report Builder 展示数据的方式。
