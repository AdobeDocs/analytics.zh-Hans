# 什么是Report Builder?

Adobe Report Builder 是 Microsoft Excel 的加载项。它允许您从报告和分析数据构建自定义请求，并可将其插入到Excel工作表中。 请求可动态引用工作表中的单元格，您可以更新和自定义 Report Builder 显示数据的方式。
