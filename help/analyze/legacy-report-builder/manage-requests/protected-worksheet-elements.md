# 受保护的工作表元素

{{legacy-arb}}

锁定单元格以保护工作表时，无法创建、编辑或刷新请求。 在使用请求管理器之前，请取消工作表中的所有保护。

如果您在刷新请求过程中刷新请求并尝试编辑单元格，则会显示Excel toast以指示电子表格当前受保护。 在刷新请求流程完成后，您可以访问电子表格，并删除临时电子表格保护。
