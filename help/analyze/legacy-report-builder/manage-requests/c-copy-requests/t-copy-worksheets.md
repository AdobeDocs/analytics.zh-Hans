# 复制工作表

{{legacy-arb}}

在同一工作簿中将一个电子表格复制为另一个电子表格。

1. 右键单击电子表格中任意位置的单元格，并选择&#x200B;**[!UICONTROL 复制带有请求的工作表]**。
1. 再次右键单击并选择&#x200B;**[!UICONTROL 粘贴带有请求的工作表]**。

   新工作表将添加到您的工作簿。 默认情况下，工作表与原始工作表具有相同的名称，但具有后缀(2)。 如果重复粘贴操作，则新工作表具有后缀(3)，依此类推。 粘贴已复制的电子表格时，新电子表格将位于活动工作表的右侧（紧跟在活动工作表之后）。

   与复制和粘贴请求不同，Report Builder在复制电子表格时复制和粘贴直接输入的单元格内容（例如，列标题、行或格式的标签）。
