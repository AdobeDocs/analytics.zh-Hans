# 复制请求概述

{{legacy-arb}}

您可以复制映射到多个请求的单元格并将内容粘贴到所选的空白电子表格区域。

复制单元格后，Report Builder和Excel会确定粘贴最小复制次数所需的区域。 如果区域足够大，则粘贴操作会创建所有请求的副本，其中每个粘贴的请求都具有与原始请求相同的空间排列和格式。

这称为传播请求。 这是创建长报表的简单快速的方法。 Report Builder首先通过将所有请求粘贴到“目标粘贴”区域的单元格中来传播请求，然后根据为请求建立的报表日期刷新单元格。
