# 分类

{{legacy-arb}}

分类划分用于将分析报表数据映射到相关属性。 分类可用于多种目的，但最常用于分类内部和外部促销活动跟踪代码以及产品ID。

例如，[!UICONTROL 营销活动]报告可能具有[!UICONTROL 经理]分类和[!UICONTROL 销售机会]分类。 然后，这些分类可以按照特定的层次结构排序，并且每组报表值可以划分为子组。

您可以使用与访问报表包的其他报表相同的方式访问分类，并且维度将显示为组内的顶级报表。 如果报表具有分类，则会创建一个菜单子组，以在同一位置收集报表及其所有分类。 报告显示为分类中列表的最后一个条目。

请参阅[分类](/help/components/classifications/classifications-overview.md)。
