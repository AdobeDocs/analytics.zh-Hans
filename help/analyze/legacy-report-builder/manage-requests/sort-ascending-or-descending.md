# 升序排序或降序排序

{{legacy-arb}}

您可以按升序或降序排列请求管理器中列出的请求。

单击表标题中的列名。 列通过在表格标题中显示三角形图标来显示排序顺序。 如果三角形指向下方，则排序顺序为降序（对于字母数字数据，为9-1和Z-A）；如果指向上方，则排序顺序为升序（对于字母数字数据，为1-9和A-Z）。

默认排列顺序是升序。
