# 什么是旧版Report Builder？

{{legacy-arb}}

Adobe Report Builder 是 Microsoft Excel 的一个加载项。 通过它，可从 Adobe Analytics 数据构建自定义的请求，并可将其插入 Excel 工作表。 请求可动态引用工作表中的单元格，并且您可更新和自定义 Report Builder 展示数据的方式。
