# 配置自定义布局

{{legacy-arb}}

自定义布局提供引导布局的大多数功能，但是允许您选择网格中的每个项目应该位于电子表格中的什么位置。此布局具备了早期版本的灵活性，包括添加划分和 Microchart。

下列主题专门针对[!UICONTROL 自定义布局]：

* [创建 Microchart](/help/analyze/legacy-report-builder/layout/t-create-a-microchart.md)
