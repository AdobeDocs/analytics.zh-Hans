# 创建 Microchart

{{legacy-arb}}

在自定义布局中，您可以将量度列转换为具有相应百分比值的条形图或者具有相应条形图的百分比值。百分比值是时段内整个未过滤请求的量度值或总计。

1. 在 Excel 中，使用[自定义布局](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md)创建请求。
1. 右键单击量度值列。
1. 选择“**[!UICONTROL 更改输出显示]**”，然后选择您希望看到的 Microchart 类型和百分比显示。

   Report Builder会更改列中的显示以匹配您的选择。
