# 删除输出映射

{{legacy-arb}}

如果您正在编辑工作簿，并且希望为新请求腾出空间，可能需要删除请求。

腾出空间的唯一方法是删除映射到单元格的请求。 如果通过删除行或列来删除电子表格中的请求，则基础请求的有效性将丢失（并在刷新时失败）。 Excel菜单选项&#x200B;**[!UICONTROL 编辑]** > **[!UICONTROL 清除内容]**&#x200B;将删除单元格中显示的值。 可以通过刷新单元格的内容来恢复内容。

要删除电子表格单元格的特定映射，请找到行、列或量度项，然后单击&#x200B;**[!UICONTROL 删除]**。
