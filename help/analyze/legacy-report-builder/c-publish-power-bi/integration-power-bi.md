# 适用于 Power BI 的 Adobe Analytics 内容包

{{legacy-arb}}

Adobe Analytics内容包提供预建的Power BI功能板和一组Power BI报表，这些报表提供关于网站流量和用户维度的洞察。 您可以使用提供的功能板和报表，也可以对其进行自定义以突出显示您最关心的信息。 数据每天自动刷新一次。

有以下三个内容包可用：

* Adobe Analytics
* Adobe Analytics移动设备应用程序Analytics
* Adobe Analytics流量分析

有关更多信息，请参阅[适用于 Power BI 的 Adobe Analytics 内容包](https://powerbi.microsoft.com/zh-cn/documentation/powerbi-content-pack-adobe-analytics/)。
