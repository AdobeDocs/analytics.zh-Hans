# 将数据手动导入 Power BI

{{legacy-arb}}

如果您要通过 Power BI 手动导入 Analytics 数据，请按照以下说明执行操作。

1. 在Power BI中，单击左下角屏幕中的&#x200B;**[!UICONTROL 获取数据]**。
1. 在&#x200B;**[!UICONTROL 导入或连接到数据]** > **[!UICONTROL 文件]**&#x200B;下，单击&#x200B;**[!UICONTROL 获取]**。

   ![单击“导入或连接到数据”下方的“获取”图标。](assets/get-data.png)

1. 单击“本地文件”。

   ![单击“本地文件”图标。](assets/local-file.png)

1. 选择要上载的文件，然后单击&#x200B;**[!UICONTROL 打开]**。
1. 单击&#x200B;**[!UICONTROL 上传Excel文件至Power BI]**&#x200B;下的&#x200B;**[!UICONTROL 上传]**。

   ![单击“上传”以上传您的 Excel 文件。](assets/upload-excel-file.png)

1. 应当会显示消息“您的文件已上载”。
