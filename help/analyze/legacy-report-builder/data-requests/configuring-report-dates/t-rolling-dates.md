# 滚动日期

{{legacy-arb}}

为请求自定义滚动日期：

1. 在“[!UICONTROL 请求向导: 第 1 步]”中，选择&#x200B;**[!UICONTROL 滚动日期]**。
1. 单击&#x200B;**[!UICONTROL 滚动]**&#x200B;链接以选择滚动类型，然后使用日期链接配置滚动日期。

   “[!UICONTROL 滚动日期]”的设置根据您的选择而发生变化。例如，当您选择[!UICONTROL 每周滚动]时，Report Builder会提供特定于周的设置，您可以选择这些设置作为滚动模式。

1. 根据需要配置高级选项。
