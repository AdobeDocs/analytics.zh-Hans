# 请求向导定义 — 固定日期

{{legacy-arb}}

Report Builder 中固定日期的定义。

| 字段 | 定义 |
|--- |--- |
| 此报表包 | 使保存的日期范围仅可用于当前报表包。 |
| 所有报表包 | 使保存的日期范围可供登录用户访问的所有报表包。 |
| 输入参数名称 | 键入已保存日期的名称。 您可以将此日期用作Report Builder中的模板。 此名称会显示在您可以应用已保存日期的任何位置。 更改模板名称不会影响先前创建的数据请求中使用的日期配置。 |
