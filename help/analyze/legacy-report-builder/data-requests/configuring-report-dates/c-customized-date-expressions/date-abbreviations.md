# 日期缩写

{{legacy-arb}}

时间缩写是指期间开始时的某个特定日期，而不是某个天数范围。 天数范围通过为两个时间缩写或术语指定开始日期和结束日期来表示。

有关使用正确语法的信息，请参阅[语法说明](/help/analyze/legacy-report-builder/data-requests/configuring-report-dates/c-customized-date-expressions/examples-of-date-ranges-using-customized-expressions.md#section_555D6563B2D94FA3BDD801DC0B8C289D)。

* cd =当前日期（今天）
* cw =本周（本周的第一天）
* cm =当月（当月的第一天）
* cq =当前季度（当前季度的第一天）
* cy =当前年份（当前年份的第一天）

在以下列表中，*[UNIT]* 表示任意整数乘数。

* [UNIT] d = 天
* [UNIT] w = 周
* [UNIT] m = 月
* [UNIT] q = 季度
* [UNIT] y = 年
