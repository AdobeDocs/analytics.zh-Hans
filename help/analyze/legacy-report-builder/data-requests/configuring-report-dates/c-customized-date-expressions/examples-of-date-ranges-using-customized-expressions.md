# 使用自定义表达式的日期范围示例

{{legacy-arb}}

有关使用以自定义表达式表示的日期范围的示例、注释和语法说明。

该表假定当前日期为2011年11月10日星期一（使用公历）。

| 示例 | 日期范围 | 自定义表达式 | 报告的日期范围 |
|---|---|---|---|
|  | | **来自** | **至** |
| 1 | 两周前 | `cw-2w  \| cw-1w-1d` | 10月26日至11月1日 |
| 2 | 去年第五个月的前3天 | `cy-1y+4m  \| cy-1y+4m+2d` | 2010年5月1日至5月3日 |
| 3 | 一整周，从4周前开始 | `cw-4w  \| cw-3w-1d` | 10月12日至10月18日 |
| 4 | 上一年中的最后一周 | `cw-53w  \| cw-52w-1d` | 2010年11月至11月9日 |
| 5 | 一个月从2个月前开始 | `cm-2m  \| cm-1m-1d` | 9月1日至9月30日 |
| 6 | 12个月前（去年） | `cm-12m  \| cm-11m-1d` | 2010年11月1日至11月30日 |

## 示例说明 {#section_37801B0D6D364ABAA8DCE3A4C0123B2C}

**示例 1**

如果今天是2011年11月10日星期一，则获取当前日期并减去一周即可获得10月的最后一整周。

**示例 2**

将四个月添加到年初（1月份）以获取5月份；将两天添加到月份的第一天以获取该月的第三天。

## 语法说明 {#section_555D6563B2D94FA3BDD801DC0B8C289D}

可通过将两个术语与运算符关联来创建涵盖大多数日期范围的自定义表达式。 术语是整数乘数和句点缩写的组合。 术语的一个示例为18d。 运算符的一个示例为+。

* 运算符和术语之间不允许使用空格。
* 仅使用以下缩写：cd cw cm cq cy d w m q y
* 最佳做法是在开始日期和结束日期中使用相同的日期引用：cd、cd或cw、cw或cy、cy。 混合日期引用可能会导致在一年中的某些时间出现无效日期。
* d w m q y的有效倍数由在缩写前面的整数( 1 2 3 ... )形成，如53d 3w 5q 9m 2y
* 不允许使用非整数。
* 不要在缩写前面只加零。 例如，不允许0w。
* 以下运算符用于连接缩写：+ -
* 由于日期范围必须相对于当前期间进行计算，因此表达式中的第一个词始终以c开头。
