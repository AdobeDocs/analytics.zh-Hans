# 选择一个报表包

{{legacy-arb}}

帮助您在 Excel 中选择报表包的步骤。

1. 在 Excel 中，单击&#x200B;**[!UICONTROL 外接程序]**。
1. 在工具栏中，单击&#x200B;**[!UICONTROL 创建]**。
1. 在“[!UICONTROL 请求向导: 第 1 步]”窗口的“[!UICONTROL 报表包]”下，选择一个报表包。
