# Adobe Analytics入门

欢迎观看Adobe Analytics数字体验入门系列！

Adobe Analytics 使组织能够从任何数字客户交互中收集数据并获得可操作洞察。通过深入分析、多用途报表和预测智能，组织可以获得构建更好的客户体验所需的洞察信息。

您的Adobe Analytics历程从载入正确的配置和用户访问权限开始。 以下资源可简化该入职流程，并使您和您的组织更轻松地成功使用Adobe Analytics。

观看以下视频，完成从Adobe Admin Console为用户提供的入门培训历程，访问设置到Adobe Experience League以获取文档和支持，最后访问Adobe Analytics本身。

## Adobe Experience League

## Adobe Admin Console

>[!VIDEO](https://video.tv.adobe.com/v/3476921/?learn=on&enablevpops)

加入[Adobe Admin Console](https://adminconsole.adobe.com/)以管理贵组织的产品权利。 添加、删除或修改用户访问权限。

## Adobe Admin Console用户管理

>[!VIDEO](https://video.tv.adobe.com/v/3476898/?learn=on&enablevpops)

加入[Adobe Admin Console用户管理](https://adminconsole.adobe.com/redirect@AdobeOrg/users)以管理用户、组和配置文件。 了解如何将产品角色分配给用户或组。

## 提出支持案例

>[!VIDEO](https://video.tv.adobe.com/v/3476787/?learn=on&enablevpops)
