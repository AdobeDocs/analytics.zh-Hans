# 启动 Activity Map{#launch-activity-map}

假设您已经安装了正确的浏览器插件，那么就可以直接通过网页启动 Activity Map。

1. 单击浏览器地址栏右侧的Adobe Analytics图标：\
   ![](assets/an_icon.png){width="90px"}
1. 使用您的用户名和密码登录公司网页。
1. 此时，该应用程序将会启动，并显示工具栏和叠加图。
