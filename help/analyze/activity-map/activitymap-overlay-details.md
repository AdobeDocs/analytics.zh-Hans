# 叠加图的详细信息

当您将鼠标悬停在链接叠加图上方时，可以显示出叠加图的详细信息。

叠加图的详细信息可以显示针对该链接所跟踪的下列值：

* 指标
* 原始值
* 排名
* 百分比值
* 链接 ID
* 区域
* 显示在“页面上的链接”报表中的内容
