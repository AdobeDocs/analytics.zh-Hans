# 客户分段

允许您在 Activity Map 中应用多区段过滤器。

>[!NOTE]
>
>区段控件仅在标准模式下可用，因为Adobe Analytics平台不支持对实时数据进行分段。

您可以从下面显示的&#x200B;**[!UICONTROL 区段]**&#x200B;下拉菜单中选择一个或多个区段。对于所选的报表包，列出的区段与您拥有的区段或与您共享的区段相同。

![](assets/segments.png)

当区段选择发生更改时，并且在关闭区段选择器后，将会为应用程序中显示的所有量度启动一个新的Analytics数据检索流程。 当从一个页面导航至另一个页面时，所选择的区段会被记录下来。

当选择多个区段时，区段选择标签（处于折叠状态）将显示“多个”。当您点击控件时，将显示完整的选定区段列表。

![](assets/two_segments.png)

选定区段将被应用于 Activity Map 中存在的所有 Analytics 报表，例如叠加图、**[!UICONTROL 页面上的链接]**&#x200B;和&#x200B;**[!UICONTROL 详细信息页面]**&#x200B;报表。
