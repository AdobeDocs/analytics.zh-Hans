# 热键（快捷键）

Analysis Workspace 中提供了热键（键盘快捷键），用于无缝导航、分析、添加面板和可视化效果以及实现民主化（共享）。

注释：
* `Ctrl` 针对的是 Windows，`Command` 针对的是 Mac
* `Alt` 针对的是 Windows，`Option` 针对的是 Mac

## 导航操作

| 操作 | 快捷键 |
| --- | --- |
| 新建项目 | Ctrl+Shift+P |
| 打开现有项目 | Ctrl+O |
| 转到“面板”窗格 | Alt + Shift + 1 |
| 转到“可视化”窗格 | Alt + Shift + 2 |
| 转到“组件”窗格 | Alt + Shift + 3 |
| 折叠/展开所有面板 | Alt + m |
| 折叠/展开活动面板 | Alt + Ctrl + m |
| 移到下一个面板 | Alt + 向右键 |
| 移到上一个面板 | Alt + 向左键 |

## 分析操作

| 操作 | 快捷键 |
| --- | --- |
| 搜索左边栏 | Ctrl + / |
| 创建警报 | Ctrl+Shift+A |
| 创建注释 | Ctrl + Shift + O |
| 创建计算量度 | Ctrl+Shift+C |
| 创建日期范围 | Ctrl+Shift+D |
| 创建区段 | Ctrl+Shift+E |
| 全部清除 | Alt + w |
| 复制 | Ctrl+C |
| 重做 | Ctrl+Shift+Z |
| 还原 | Ctrl+Z |
| 刷新组件 | Alt + Shift + r |
| 刷新项目 | Alt + r |
| 设置为登录页面 | Ctrl+Shift+I |
