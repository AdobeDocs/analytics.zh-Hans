# 模板(*现在称为“报表”*)

模板不再用作预建工作区项目的术语。 报告是指Adobe预建的任何内容，例如Reports &amp; Analytics中的报告（生命周期结束）和工作区中的预建报告。 但是，术语 *模板* 有时用于引用您的公司创建的模板或报表。 自定义模板称为 *自定义公司报表*.

有关创建Workspace报告和创建自定义公司报告的信息，请参阅以下文档：
