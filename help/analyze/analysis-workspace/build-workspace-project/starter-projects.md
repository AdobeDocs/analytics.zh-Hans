# 模板（*现在称为报表*）

模板不再用作预建 Workspace 项目的术语。报表是指 Adobe 预建的任何内容，例如 Reports &amp; Analytics（已停用）中的报表和 Workspace 中的预建报表。不过，术语&#x200B;*模板*&#x200B;有时用于指您公司创建的模板或报表。自定义模板称为&#x200B;*自定义公司报表*。

有关创建 Workspace 报表和创建自定义公司报表的信息，请参阅以下文档：
