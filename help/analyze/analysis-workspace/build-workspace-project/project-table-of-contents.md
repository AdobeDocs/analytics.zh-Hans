# 项目目录

每个项目中都有一个目录。 这允许您在项目中的面板和可视化图表之间快速切换。 在查看包含许多面板和可视化图表的较大型项目时，此功能特别有用。

要查看项目的目录，请执行以下操作：

1. 在Analysis Workspace中，转到要查看目录的项目。

1. 在左侧导航中，选择目录图标 ![目录图标](assets/toc-icon.png).

   此时将显示并展开项目的目录。

1. 单击目录中的部分可跳转到项目中的面板或可视化图表。
