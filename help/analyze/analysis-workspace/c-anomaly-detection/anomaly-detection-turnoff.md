# 关闭异常检测

您可以在列级别关闭异常检测。

1. 在列中选择![设置](/help/assets/icons/Setting.svg)。
   ![](assets/turn-off-anomalies.png)

1. 取消选择&#x200B;**[!UICONTROL 显示异常]**。
