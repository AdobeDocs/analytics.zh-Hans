# 添加组件描述

Analytics管理员可以在报表包中为组件(Dimension、量度、区段和日期范围)添加描述。

在Analysis Workspace中构建项目或查看报表时，您组织中的用户可以查看相关描述。 描述可帮助用户了解如何最好地使用各种组件。
