# 散点图

散点图可视化显示了维度值与最多3个指标之间的关系。 它需要三个组件，并支持最多四个组件的可视化。

* 行组件（通常为尺寸）表示图形上的每个点。 不同的行显示为不同的彩色点。
* 最左侧的列（通常为度量）绘制点在Y轴上的位置（垂直）。
* 第二列绘制点在X轴上（水平）的位置。
* 第三列确定点的半径。
* 散点图可视化将忽略自由格式表中的所有后续列。
