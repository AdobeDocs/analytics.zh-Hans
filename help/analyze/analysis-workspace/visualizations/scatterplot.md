# 散点图

显示所提供的展示次数以及有多少名独特用户已看到了这些展示。

每个数据点的大小以可视化方式提示查看者见到某广告的平均次数。大小和数据依据您选择的维度、日期范围和过滤器而变化。

![](assets/scatter.png)

>[!NOTE]
>
>与散点图关联的表至少需要两列。第一列定义 X 轴，第二列定义 Y 轴。如果存在第三列，那么散点图图表可使用它来决定点的半径。换言之，列 *1*、*2* 和 *3* 映射到 *X*、*Y* 和&#x200B;*点半径*。
