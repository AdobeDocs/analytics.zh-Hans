# 项目符号图表

让您可以查看您感兴趣的值与其他性能范围（目标）进行比较或衡量的结果。

![](assets/bullet-image.png)

项目符号图具有单一的主要衡量标准（例如，当前年初至今的收入），并允许您输入定性范围的绩效范围（例如，与目标收入相比）。 可指定高、中和低。 您可以在[!UICONTROL 可视化设置]中指定目标范围。
