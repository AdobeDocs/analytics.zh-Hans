# 树状图

将分层（树结构）数据显示为一组嵌套的矩形。

树上的每个分支都被给予一个矩形，然后为其贴上代表子分支的更小矩形。

![](assets/treemap.png)

当颜色和大小维度以某种方式与树结构关联时，用户通常可以轻松地查看平时很难以其他方式见到的模式，例如当有某个颜色特别重要时。树形图的第二个优势在于，它们可以通过精心的构建有效地利用空间。
