# 管理警报

您可以在警报管理器中管理现有警报。 您可以对警报执行各种管理任务，如标记、重命名、删除等。

有关在警报管理器中管理现有警报的信息，请参阅[警报管理器](/help/components/c-alerts/alert-manager.md)。
