# 区段比较中使用的统计测试

每个排名最前的比较表都会显示差异分数。 该得分由若干统计测试根据所做的比较来计算。 但是，无论使用哪种测试，差异分数都显示为0到1之间的值。

0分表示两个区段之间没有差异，1分表示两个区段之间存在很大差异。 有两种类型的统计测试用于生成这些差异分数：

* 对于&#x200B;**[!UICONTROL 热门量度]**&#x200B;表，使用Mann-Whitney U测试，
* 对于&#x200B;**[!UICONTROL 排名最前的Dimension项目]**&#x200B;和&#x200B;**[!UICONTROL 排名最前的区段]**&#x200B;表，使用风险差异比较。

## 热门量度差异分数

在“顶级量度”表中，区段比较工具使用两个示例Mann-Whitney U测试。 此测试为非参数等式测试，用于比较每个指标对所考虑区段的一维概率分布。 量度表中的差异分数是计算的U统计量（表示两个区段在特定量度中的随机差异）的p值和观察到的差异的相对幅度的组合。 大差异分数（接近1）意味着特定量度具有大的相对差异，并且对于区段不同的统计置信度很高。

## 主要维度项目和排名最前的区段差异分数

要计算“排名最前的Dimension项目”表和“排名最前的区段差异”表的差异分数，请使用相对风险差异算法（与风险比率类似，不过使用的是差异而非比率）。 计算风险差异的方法是，将一个选定区段的维度项目（或与区段表中的区段重叠）的累计关联减去另一个选定的区段。 高差异分数（接近1）意味着特定维度项目或第三区段在一个选定区段中非常突出，而在另一个区段中不是非常突出。

>[!NOTE]
>
>在所有三个表中，差异统计都基于适当的访客取样，这样既可以尽可能快速地运行统计过程，又可以保持统计学上的准确性。 尽管差异分数是基于样本的，但表格中提供的结果不是样本的。 为确保统计显着性，每个统计测试都依赖于动态分配算法，以便较小的区段包含的样本大小提供不到3%的错误边距。 如果区段包含非常少的访客（少于1,000），则使用所有可用数据而不是样本来计算差异分数。
