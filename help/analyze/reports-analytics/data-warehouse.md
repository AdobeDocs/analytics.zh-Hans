# 数据仓库请求

Data Warehouse 是指用于存储的、未经处理的原始数据副本，以及可通过过滤数据而运行的自定义报表。您可让报表根据您的特定问题显示原始数据的高级数据关系。与报表界面中显示的大多数报表不同，Data Warehouse 报表通过电子邮件或 FTP 发送，且数据处理时间最长为 72 小时。处理时间取决于查询的复杂程度和请求的数据量。
