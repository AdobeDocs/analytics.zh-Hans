# Analytics 工具指南

![横幅](../../assets/doc_banner_analyze.png)

本指南重点介绍了可以在 Adobe Analytics 中使用的所有分析工具。

* **Analysis Workspace：** Adobe Analytics 中最重要的功能。Workspace 提供了一个画布，您可以在其中拖动组件以满足报表需求。
* **活动图：**&#x200B;作为一个浏览器插件，它可以在您的网站上显示一个覆盖层，以展示点击量最多的元素。
* **Ad Hoc Analysis：**&#x200B;一款具有强大报告功能的独立 Java 应用程序。这款工具不再更新，并且最终将会停用。
* **报表 API：**&#x200B;可直接向 Adobe 的服务器提出报表查询请求，并且获得的响应结果可以在您自己的客户报表工具中使用。
* **Report Builder：**&#x200B;一种 Excel 加载项，可允许您检索 Analytics 数据，并将其直接放入工作簿中。
* **Reports &amp; Analytics：**&#x200B;一个包括大量预建报表的工具。Adobe 建议您使用 Analysis Workspace 以满足大多数的报表需求。
