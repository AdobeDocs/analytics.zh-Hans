# Adobe 集成：夜间数据导入{#adobe-integration-nightly-data-import}

集成的数据收集部分可收集有关网站访客的点进和显示到达数据。为获取 DFA 点击、展示和成本量度，Google 与 Adobe 协同提供一种夜间流程，用于将这些额外数据导入集成的报表包。这些量度是通过数据源导入的，这意味着它们仅在汇总中可用，而不是在访问级别。请参阅[集成功能](../dfa-data-connector-analytics/dfa-integration-features.md#concept-ff93289d1662410e98f62c200394b3e3)，以获取更多详细信息。
