# 集成先决条件{#integration-prerequisites}

列出Adobe Analytics客户的先决条件。

## Adobe客户先决条件 {#section-23ba9dd1a1774f3099a6764d3fadd4ef}

* 必须是Adobe Analytics的当前客户。
* 必须具有管理员权限。
* 报表包中必须有个可用的和启用的eVar。
* 必须在您的报表套件中启用个可用活动。
