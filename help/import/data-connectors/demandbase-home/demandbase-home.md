# Demandbase Data Connector for Adobe Analytics{#demandbase-data-connector-for-adobe-analytics}

此Adobe®数据连接器集成将Demandbase的实时ID服务与Adobe Analytics的行为信息相结合，为B2B组织创建了强大的分析、优化和内容个性化机会。

根据预先确定的特征向潜在客户提供相关内容，推动更合格的潜在客户并提高转化率。交付与特定行业、公司规模或特定公司本身相关的内容允许您直接向访客发送正确的信息，而不依赖于他们在您的网站上找到它。这将带来更少的弹回次数和更健康的转化漏斗。
