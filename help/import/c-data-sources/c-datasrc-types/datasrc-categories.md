# 数据类型和类别概述

数据源类别可识别具有相似功能的不同数据源类型。

类别提供了一种从用户角度对数据源进行分组的方法。 通过数据源 UI 创建数据源时，首先要选择一个数据源类别，然后选择一个特定的数据源类型。每个类别都包含多种支持相似数据类型的数据源类型。数据源具有以下数据源类别：
