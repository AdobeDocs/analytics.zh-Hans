# 将Google Display广告数据与Adobe Analytics

此集成允许您将客户行为与Google AdWords中的活动和成本数据关联。

1. 登录Google活动管理器。
2. 设置展示广告的重复导出，以便您上传展示广告指标（展示次数、点击次数、成本）的增量，按最细的ID（广告ID）进行划分。
3. 上传完整的广告ID元数据集，如活动、网站等。
4. 使用所有元数据对广告ID进行分类。
5. 将它们格式化为可供数据源和分类使用的格式。
