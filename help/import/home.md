# Analytics 导入指南

![横幅](../../assets/doc_banner_import.png)

本指南介绍如何将数据引入 Analytics。其中包括：

* **[数据源](c-data-sources/datasrc-home.md)**:将文件上传到指定的Adobe FTP站点。 Adobe 将检索该文件，并包含您的报表包中的数据。
* **[数据插入API](c-data-insertion-api/c-data-insertion-api.md)**:通过API将数据直接发送到Adobe的数据收集。
* **[数据连接器](data-connectors/getting-started-data-connectors.md)**:与受支持的第三方合作，将他们的数据导入Adobe Analytics中，以丰富您的报告套件。
