# Analytics 导入指南

![横幅](../../assets/doc_banner_import.png)

本指南介绍如何将数据引入 Analytics。其中包括：

* **数据源：**&#x200B;将文件上传到指定的 Adobe FTP 站点。Adobe 将检索该文件，并包含您的报表包中的数据。
* **数据插入 API：**&#x200B;通过 API，将数据直接发送到 Adobe 的数据收集中。
* **Data Connectors：**&#x200B;与受支持的第三方合作，将其数据引入 Adobe Analytics 以丰富您的报表包。
