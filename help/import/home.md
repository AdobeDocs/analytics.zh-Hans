# Analytics 导入指南

![横幅](../../assets/doc_banner_import.png)

本指南介绍如何将数据引入Analytics。该服务包括：

* **数据源：** 将文件上传到指定的Adobe FTP站点。Adobe将检索该文件并包含您的报表套件中的数据。
* **数据插入API：** 通过API将数据直接发送到Adobe的数据收集。
* **数据连接器：** 与受支持的第三方合作，将其数据引入Adobe Analytics以丰富您的报表套件。
