# 使用SFTP将Adobe数据发送到外部FTP帐户

Adobe工程服务组可以设置自定义项目，将您的数据重定向到使用SFTP的外部FTP帐户。

它可以在无需从 Adobe FTP 站点内检索数据的情况下，将您的数据安全地发送给您。要确定该项目的范围，请联系您的 Adobe 客户经理。
