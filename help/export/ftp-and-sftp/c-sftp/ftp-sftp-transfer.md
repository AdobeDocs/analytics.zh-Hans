# 通过 SFTP 将 Adobe 数据发送到外部 FTP 帐户

Adobe工程服务小组可以设置自定义项目，以使用SFTP将数据重定向到外部FTP帐户。

它可以在无需从 Adobe FTP 站点内检索数据的情况下，将您的数据安全地发送给您。要确定该项目的范围，请联系您的 Adobe 客户经理。
