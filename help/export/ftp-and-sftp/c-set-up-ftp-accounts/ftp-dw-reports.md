# Data Warehouse 传送的报表

>[!NOTE]
>
>以下信息与FTP和SFTP目标类型相关。 FTP 和 SFTP 都是旧版目标类型。 在配置Data Warehouse报表时，您应该使用云目标类型，这样更安全。 有关为Data Warehouse报表配置云目标类型的详细信息，请参阅[创建Data Warehouse请求](/help/export/data-warehouse/create-request/t-dw-create-request.md)。

通过 Data Warehouse，可要求报表根据特定问题显示从原始数据产生的高级数据关系。

在这种情况下，FTP 传送可作为[!UICONTROL 高级传送选项]设置的一部分提供。 使用FTP时，您需要一个现有的内部FTP帐户或Adobe FTP帐户。 Adobe不会通过Data Warehouse界面自动为您设置FTP帐户。 根据文件大小和处理的报表数，应立即FTP交付。

有关FTP限制和数据保留的信息，请参阅[FTP限制和数据保留](/help/export/ftp-and-sftp/ftp-limits.md)。
