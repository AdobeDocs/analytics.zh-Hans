# 使用被动 FTP 模式

选择主动 FTP 或被动 FTP 控制如何建立端口连接，并且该选择对防火墙可能会产生一些影响。

Adobe使用被动FTP，这是一种更安全的数据传输形式，其中数据流由文件传输程序(FTP)客户端设置和启动，而不是由FTP服务器程序设置和启动。 如果连接到Adobe FTP时遇到困难，请确保使用被动模式。
