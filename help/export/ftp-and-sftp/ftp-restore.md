# 恢复已删除的 FTP 数据和 FTP 帐户

Adobe 无法恢复任何已从 FTP 帐户中删除的数据。

如果您计划使用的FTP帐户已被禁用，请联系您的Adobe客户团队，他们可以与您讨论其他选项。
