# Analytics 导出指南

![横幅](../../assets/doc_banner_export.png)

本指南概述了从Adobe Analytics获取数据的方法。该服务包括：

* **数据源：** 获得原始数据的每小时或每日导出数据。每行都是单独的点击，每一列都是一个变量。数据源通常发送到FTP站点。
* **数据仓库：** 使用请求向导检索数据电子表格输出。数据仓库使用不同的处理架构允许任意数量的行和任意数量的唯一值。
