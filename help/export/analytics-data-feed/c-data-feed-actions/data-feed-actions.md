# 数据馈送面板

您可以对数据馈送执行各种操作。

在“数据馈送”界面中，选中一个或多个馈送旁的复选框时，可用的操作会显示为列表上方的一系列按钮。

![](assets/actions.png)

要编辑馈送，请单击列表中的馈送名称。
