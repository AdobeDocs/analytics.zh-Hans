# 作业状态

作业旁可显示若干状态标签。

| 状态 | 描述 |
|---|---|
| 正在等待数据 | 正在运行作业，且正在收集数据。 |
| 失败 | 发生问题。请参阅[作业疑难解答](/help/export/analytics-data-feed/jobs-troubleshooting.md)。 |
| 就绪 | 作业随时准备运行。 |
| 正在处理 | 作业正在创建数据文件并准备发送它们。 |
| 完成 | 已创建作业并发送其文件。 |
| 正在等待导出 | 导出系统尚未将该时间段的数据打包、分类和复制到存储服务器。 |
| 无数据 | Data Warehouse 中没有应用到馈送的数据。 |
