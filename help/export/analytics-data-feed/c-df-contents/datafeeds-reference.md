# 数据列引用

使用此页面了解每个列中包含的数据。大多数实施不使用每一列，因此在确定要包含在数据源导出中的列时，可以引用此页面。

> [!IMPORTANT] 对于任何给定列(例如定义为255个字符的列)，由于字符串中添加字符转义值，数据源可以发送其他字符。如果您的实施定期发送超过字符限制的值，请记住此主题。
