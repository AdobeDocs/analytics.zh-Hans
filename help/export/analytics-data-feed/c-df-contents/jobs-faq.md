# 作业常见问题解答

***哪些因素可能会影响每小时数据馈送的提交时间？***

以下几个因素可能会影响每小时数据馈送的提交时间，包括：

* 报表包延迟（即未公布的流量尖峰）
* 上行处理
* 高峰和非高峰时间
* Internet 连接速度
