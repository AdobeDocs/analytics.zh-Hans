# 移动属性查找

移动属性显示点击中移动设备的属性。 此文件在以下情况下显示：

* 已启用动态查找。 您组织中的支持代表可以使用所需的源ID联系客户关怀以启用动态查找。
* 包 `mobile_id` 含该列。
* 列 `user_agent` 被排除。 由于与DeviceAtlas签订了许可协议，因此需要排除此漏洞。
