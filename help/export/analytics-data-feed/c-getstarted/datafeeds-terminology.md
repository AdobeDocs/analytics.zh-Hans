# 术语

使用 Analytics 数据馈送界面时，了解关键术语十分重要。

本指南中的大多数术语是在上下文中定义的。但是，了解下列术语将会帮助你更好地阅读本文档。

*`Data Feed (capitalized)`*：Adobe 数据馈送服务及其界面。

*`data feed (lowercase)`*：一组定义的参数和值，可用于收集数据集。

*`job`*：有关运行和导致提交一个或多个含数据文件的数据馈送的示例。
