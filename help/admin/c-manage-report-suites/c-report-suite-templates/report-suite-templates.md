# 报表包模板概述

报表包模板可为几种类型的报表包配置最常见的设置。

若要节省创建新报表包的时间，您可以选择与您设计的报表包配置类似的模板。选择报表包模板不会在任何方面限制您的实施。创建报表包后，您可以对使用报表包模板完成的任何配置进行更改。
