# Analytics 管理指南

![横幅](../../assets/doc_banner_admin.png)

Adobe Analytics目前有两个管理员区域：

* Adobe Admin Console：使用此区域可提供Experience Cloud工具和管理用户权限。它位于 [adminconsole.adobe.com](https://adminconsole.adobe.com)。
* Analytics Admin Console：将此区域用于报表套件和变量管理。可通过单击Adobe Analytics顶部标题中的管理员来访问它。

本指南涵盖：

* 在Analytics Admin Console中完成的所有任务。此区域包括设置报表包、变量、分类或数据管理。有关可用的报表套件和公司设置列表，请参阅 [管理工具](admin/c-admin-tools.md) 。
* 在Adobe Admin Console中完成的所有特定于Analytics的任务。此区域包括产品供应和用户权限管理。有关可在Adobe Admin Console中完成的操作的列表，请参阅 [Adobe Admin Console](admin-console/home.md) 中的Adobe Analytics。

本指南不涵盖Adobe Admin Console提供的许多通用功能。请参阅Enterprise用户指南中的 [Admin Console](https://helpx.adobe.com/enterprise/using/admin-console.html) 。

## 关键文章
