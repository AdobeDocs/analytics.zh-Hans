# Report Builder 报告

管理分配给 Report Builder 用户的许可证。

分配给用户的每个 Report Builder 许可证都会自动分配十个计划任务。 可以调整和重新分配此数量的任务。 计划任务不能超过每个用户的可用任务数。 任何超计划任务都不会运行。

您可以通过在 [Adobe Admin Console](/help/admin/admin-console/home.md) 中管理产品配置文件来向任何 [!DNL Adobe Analytics] 用户授予 Report Builder 访问权限。
