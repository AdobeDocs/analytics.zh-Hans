# 自定义报表描述

您可以为eVar、prop、分类和事件提供自定义报表描述。 所有最终用户都可以看到这些描述。

**[!UICONTROL Analytics]** > **[!UICONTROL 管理员]** > **[!UICONTROL 报表包]** > **[!UICONTROL 编辑设置]** > **[!UICONTROL *变量&#x200B;*]**

>[!NOTE]
>
>对于大多数传统分类可添加描述，但目前对于移动分类无法添加描述。

![](/help/admin/tools/assets/report_descriptions.png)

1. 提供报表的描述。 最大字符数为255。
1. 单击&#x200B;**[!UICONTROL 保存]**。

该描述显示在两个位置：

* 在该报告的标题栏中：

  ![](/help/admin/tools/assets/report_description_2.png)

* 当您将鼠标悬停在左侧导航中的报表上时，作为工具提示：

  ![](/help/admin/tools/assets/report_description_3.png)
