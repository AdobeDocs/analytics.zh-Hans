# 代码管理器

代码管理器可让您下载适用于 Web 和移动平台的数据收集代码。

**[!UICONTROL Analytics]** &gt;管 **[!UICONTROL 理员]** &gt;代 **[!UICONTROL 码管理器]**。

下载库后，您必须配置代码，以便将数据发送到正确的跟踪服务器和报表包。实施用户指南中提供了其 [他实施](/help/implement/home.md) 资源。
