# 隐私设置

您可以启用浏览器 Cookie 的隐私设置。

请参 [阅核心服务用户指南中](https://docs.adobe.com/content/help/en/core-services/interface/ec-cookies/browser-cookie-settings.html) ，为浏览器cookie启用隐私设置。
