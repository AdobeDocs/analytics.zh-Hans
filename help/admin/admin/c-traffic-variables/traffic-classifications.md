# 流量分类

流量分类允许您对流量变量 (prop) 进行分类。流量分类只能使用文本分类。

使用“流量分类”页面可以创建所选报表包的分类。分类后，任何可使用关键数据生成的报表也可通过关联的属性来生成。

在启动分类后，使用[分类导入器](/help/components/classifications/importer/c-working-with-saint.md)来对适当分类分配特定值。
