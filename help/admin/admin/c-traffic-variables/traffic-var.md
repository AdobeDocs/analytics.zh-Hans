# 流量变量 (prop) 概述

自定义分析流量变量（或 prop）可让您将自定义数据与特定流量相关事件进行关联。prop 变量会嵌入网站每个页面的执行代码中。

使用“[!UICONTROL 流量变量]”页面可以启用、禁用或重命名流量变量报表。一旦禁用了流量变量报表，它便不会显示在用户界面中。

请参阅参阅实施用户指南中的[属性](../../../implement/vars/page-vars/prop.md)。
