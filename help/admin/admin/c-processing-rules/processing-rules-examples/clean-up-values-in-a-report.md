# 清理报告中的值

您可以根据常见的错误拼写形式匹配值并更新这些值，以便它们能在报表中正确显示。

为避免意外匹配其他值，请使用最严格的匹配选项。您可以根据变量（下例中的 prop1）运行报表并搜索您选择替换的术语，确保它没有匹配不需要的值。字符串比较区分大小写。

| 规则集 | 数值 |
|---|---|
| 条件 | 如果 prop1 以 Shopping 开头 |
| 操作 | 将 prop1 的值覆盖为自定义值 Shopping |
