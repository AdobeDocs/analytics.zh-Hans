# 查看处理规则历史记录

可在历史记录中查看处理规则的更改。

1. 单击 **[!UICONTROL Analytics]** &gt; **[!UICONTROL 管理员]** &gt; **[!UICONTROL 报表包]**。
1. 选择报表包。
1. 单击&#x200B;**[!UICONTROL 编辑设置]** &gt; **[!UICONTROL 常规]** &gt; **[!UICONTROL 处理规则]**。

1. 在&#x200B;**[!UICONTROL 处理规则]**&#x200B;页面，单击&#x200B;**[!UICONTROL 查看历史记录]**&#x200B;选项卡。
