# 媒体管理

您可以启用Analytics中的媒体以保留一组特殊的媒体解决方案变量，以用于测量和报告。 这些变量的行为与自定义Evar、事件和Prop类似，但不会计入变量限制。 启用后，您的用户将在Analytics菜单中看到一组新的媒体报表。

**[!UICONTROL Analytics]** > **[!UICONTROL 管理员]** > **[!UICONTROL 报表包]** > **[!UICONTROL 编辑设置]** > **[!UICONTROL 媒体管理]**.

有关更多信息，请参阅 [适用于流媒体的 Adobe Analytics指南](https://experienceleague.adobe.com/docs/media-analytics/using/media-overview.html?lang=zh-Hans).
