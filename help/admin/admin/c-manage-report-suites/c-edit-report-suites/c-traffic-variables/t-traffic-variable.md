# 启用流量变量报告

添加并启用流量变量报告：

1. 单击 **[!UICONTROL Analytics]** > **[!UICONTROL 管理员]** > **[!UICONTROL 报告包]**，然后选择一个报告包。
1. 单击&#x200B;**[!UICONTROL 编辑设置]** > **[!UICONTROL 流量]** > **[!UICONTROL 流量变量]**。
1. 单击&#x200B;**[!UICONTROL 新增]**，然后命名报告。
1. 单击&#x200B;**[!UICONTROL 启用]**，然后单击&#x200B;**[!UICONTROL 保存]**。
