# 在产品概述页面中设置产品视图事件

可以基于页面值或变量中的特定值设置事件。

| 规则集 | 值 |
|---|---|
| 条件 | 如果页面名称等于产品概览 |
| 操作 | 将事件“产品查看事件”设置为自定义值 1 |
