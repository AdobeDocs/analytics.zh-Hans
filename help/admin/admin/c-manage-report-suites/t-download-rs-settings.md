# 下载报表包设置

要生成包含选定报表包所有设置的Excel电子表格，请执行以下操作：

1. 单击&#x200B;**[!UICONTROL 管理员]** > **[!UICONTROL 报表包]**。
1. 从[!UICONTROL 报表包]表格中选择一个报表包。
1. 单击&#x200B;**[!UICONTROL 下载]**。

   您可以直接打开电子表格文件，或将其保存以便日后查看。
