# 下载报告包设置

生成包含选定报告包所有设置的 Excel 电子表格：

1. 单击&#x200B;**[!UICONTROL 管理员]** > **[!UICONTROL 报告包]**。
1. 从[!UICONTROL 报告包]表格中选择一个报告包。
1. 单击&#x200B;**[!UICONTROL 下载]**。

   您可以直接打开电子表格文件，或将其保存以便日后查看。
