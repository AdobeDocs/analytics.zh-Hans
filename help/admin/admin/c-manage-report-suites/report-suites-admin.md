# 报表包管理器

报表包定义针对以下任意数据存储的完整、独立的报表：

* 单个网站（最典型）

* 单个网站的一部分或子集

* 为获得总数而合并的若干个网站数的全局区段

Analytics 解决方案会收集并报告这些数据存储。使用管理员“报表包管理器”可定义一些规则来管理报表包中数据的处理方式。

在登录到任何 Adobe Analytics 解决方案时，您需要选择一个要使用的报表包（除非您使用的是合并各报表包的统计报表包）。

要访问报表包，请执行以下操作：

单击 **[!UICONTROL Analytics]** > **[!UICONTROL 管理员]** > **[!UICONTROL 所有管理员]** > **[!UICONTROL 报表包]**

>[!NOTE]
>
>可通过&#x200B;**[!UICONTROL “Analytics”]**>**[!UICONTROL “组件]**”>**[!UICONTROL “虚拟报表包”]**&#x200B;管理虚拟报表包。请参阅[虚拟报表包文档](/help/components/vrs/vrs-about.md)。
