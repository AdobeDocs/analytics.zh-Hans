# Analytics 工具的产品配置文件权限

Analytics 工具权限项用于授予对 Adobe Analytics 中各项功能的访问权限。 Analytics 工具权限项不适用于单个报表包，而适用于整个 Adobe Analytics。

| Analytics 工具权限项 | 描述 |
|----|----|
| [!UICONTROL Activity Map] | 授予对“工具”>“Activity Map”的访问权限。 还允许用户使用Activity Map扩展。 |
| [!UICONTROL 管理工具UI访问] | 不使用 |
| [!UICONTROL Advertising Analytics 管理] | 授予对“管理员”>“所有管理员”>“Advertising Analytics”的访问权限。 |
| [!UICONTROL Analysis Workspace：另存为模板] | 授予对 Analysis Workspace 中“项目”>“另存为模板”的访问权限。 |
| [!UICONTROL 创建注释] | 允许创建注释。 |
| [!UICONTROL 计算量度创建] | 允许在所有 Analytics 功能中创建计算量度。 |
| [!UICONTROL 品牌联合] | 不使用 |
| [!UICONTROL 代码管理器] | 授予对“管理员”>“所有管理员”>“代码管理器”的访问权限。 |
| [!UICONTROL 代码管理器 - Web 服务] | 授予通过 1.4 API 访问代码管理器的权限。 |
| [!UICONTROL 公司设置UI访问权限] | 不使用 |
| [!UICONTROL 组件迁移] | 授予对“管理员”>“所有管理员”>“组件迁移”的访问权限。 |
| [!UICONTROL 当前数据] | 不使用 |
| [!UICONTROL 数据治理和隐私标签UI访问] | 授予对“管理员”>“数据管理隐私标签”的访问权限。 |
| [!UICONTROL Excel 许可用户] | 不使用 |
| [!UICONTROL 隐藏报表包] | 授予对“管理员”>“所有管理员”>“公司设置”>“隐藏报告包”的访问权限。 允许隐藏组织内的任何报告包，而不考虑是否具有报告包的访问权限。 |
| [!UICONTROL 集成（创建）] | 不使用 |
| [!UICONTROL 集成（更新）] | 不使用 |
| [!UICONTROL 集成（删除）] | 不使用 |
| [!UICONTROL Labs 访问权限] | 允许用户访问Labs和Labs中的原型。 |
| [!UICONTROL 旧版 Report Builder] | 允许用户使用Microsoft Excel中的旧版Report Builder加载项进行身份验证。 |
| [!UICONTROL 日志] | 授予对“管理员”>“所有管理员”>“日志”的访问权限。 |
| [!UICONTROL 日志 - Web 服务] | 授予通过 1.4 API 提取日志的权限。 |
| [!UICONTROL MCP访问] | 授予使用[Adobe Analytics MCP服务器](https://developer.adobe.com/analytics-mcp/docs/aa/)的访问权限。 所有用户（包括产品管理员）都需要此权限项才能使用此工具。 |
| [!UICONTROL 移动应用程序管理] | 不使用 |
| [!UICONTROL 待定操作] | 授予对“管理员”>“所有管理员”>“公司设置”>“待定操作”的访问权限。 |
| [!UICONTROL 偏好设置] | 不使用 |
| [!UICONTROL 报表包管理员UI访问权限] | 授予对“管理员”>“报表包”的访问权限，但不允许编辑任何设置或创建报表包的访问权限。 使用[报表包工具](report-suite-tools.md)权限项，有选择地允许用户编辑报表包设置。 |
| [!UICONTROL 报告活动管理器UI访问权限] | 授予对“管理员”>“报表活动管理器”的访问权限。 |
| [!UICONTROL Reports &amp; Analytics 访问权限] | 不使用 |
| [!UICONTROL 安全性] | 授予对“管理员”>“所有管理员”>“公司设置”>“安全管理器”的访问权限。 |
| [!UICONTROL 区段创建] | 允许在所有 Analytics 功能中创建区段并在功能之间共享区段。 |
| [!UICONTROL 区段发布] | 允许在创建或编辑区段时将区段设为 Experience Cloud 受众。 |
| [!UICONTROL 服务器调用使用情况] | 授予对“管理员”>“服务器调用使用情况”的访问权限。 |
| [!UICONTROL 与任何人共享项目链接] | 授予无法访问Analysis Workspace的用户对Adobe Analytics项目的只读访问权限的能力。 |
| [!UICONTROL 支持] | 授予对“管理员”>“所有管理员”>“公司设置”>“支持信息”的访问权限。 |
| [!UICONTROL 流量管理] | 授予对“管理员”>“所有管理员”>“流量管理”的访问权限。 |
| [!UICONTROL Web 服务访问] | 允许使用 API，包括使用 API 进行第三方网站身份验证和发送 API 调用。 |
| [!UICONTROL Web 服务] | 授予对“管理员”>“所有管理员”>“公司设置”>“Web 服务”的访问权限。 |
| [!UICONTROL Workspace项目访问权限] | 授予对 Analysis Workspace 的访问权限。 **需要此权限项才能使用Adobe Analytics。** |
