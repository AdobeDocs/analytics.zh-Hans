# Report Builder 报表

管理分配给Report Builder用户的许可证。

分配给用户的每个Report Builder许可证都会自动分配十个计划任务。 任务的数量可以调整并重新分配。计划任务数不得超过每位用户的可用任务数量。不会运行超过计划的任何任务。

您可以授予Report Builder对任何 [!DNL Analytics] 用户，方法是管理 [Adobe Admin Console](/help/admin/admin-console/home.md).
