# Adobe Analytics使用的IP和域

某些防火墙配置会阻止源自Adobe的数据收集服务器或负责访问数据的服务器的IP地址。 您可以使用此列表范围来更改组织的防火墙设置，以允许从组织内访问和发送数据。

>[!IMPORTANT]
>
>尽管Adobe尽力保持此文档的最新状态，但无法保证IP范围的列表保持不变。 可能的更改包括业务的增长和扩展，互联网注册需要更改Adobe的IP地址空间，或者因特网服务提供商停止运行。
